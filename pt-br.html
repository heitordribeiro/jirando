Casos de Sucesso

Atualização do Jira Server e Migração para Jira Cloud

Este projeto envolveu a atualização do Jira Server da versão 7 para 8 e do banco de dados Oracle 12 para 19, preparando o ambiente para uma migração tranquila para o Atlassian Cloud. Configurei os ambientes de QA e produção clonando servidores, criando novos bancos Oracle 19, migrando dados e validando as instâncias atualizadas do Jira. Em seguida, gerenciei a migração completa para o Jira Cloud, incluindo configuração, integração com Active Directory, ajustes de plugins, importação de dados e validação de acessos. Por fim, treinei os usuários finais no Jira Cloud e ofereci suporte pós-migração durante o período de operação assistida.

Implementação do Jira Service Management, Integração com Confluence e Desenvolvimento de Relatórios no Power BI

O objetivo deste projeto foi reestruturar o Jira Service Management de um cliente cujo setup anterior era desorganizado. Inicialmente, realizei o mapeamento de processos com stakeholders e coleta de dados. Configurei workflows, tipos de solicitações, permissões, integrações e automações do JSM. O trabalho incluiu configuração completa do Jira Cloud, integração com Active Directory, ajustes de plugins, migração de dados e validação de acessos. Também integrei o Confluence para construção de uma base de conhecimento, treinei usuários finais e ofereci suporte pós-implementação. Além disso, desenvolvi relatórios no Power BI conectados ao sistema ITSM, entregando dashboards com métricas-chave, como desempenho de SLA, backlog e volume de tickets, para gestores e stakeholders.

Integração entre Jira Service Management e Jira Cloud

Este projeto envolveu o planejamento e a implementação de uma integração entre Jira Service Management (JSM) e Jira Cloud, focada no tipo de solicitação de Change. Levantei requisitos junto aos stakeholders, mapeei o processo de gerenciamento de mudanças e defini parâmetros de automação para otimizar a criação e sincronização de tickets. Em um ambiente sandbox, configurei workflows para que solicitações de mudança no JSM gerassem automaticamente tickets vinculados em projetos Jira Cloud, com sincronização bidirecional de campos como status, prioridade e datas de vencimento. Após testes de aceitação iterativos e ajustes, a solução foi implantada em produção, proporcionando uma integração escalável que aumentou a eficiência, reduziu esforços manuais e deu visibilidade unificada do progresso das mudanças aos stakeholders.
